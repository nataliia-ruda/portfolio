Harmony Clinic

A multi-page commercial website concept created for a fictional women’s health clinic. This front-end-only project was built using HTML, CSS, and JavaScript, without server-side logic or a database. It focuses on responsive design, usability, and interactive features.

Features

  • Multi-page clinic website layout
  • Landing page with clear structure and navigation
  • Image galleries
  • Responsive design
  • Service rating form
  • “Call back” request form
  • Weather widget integrated with the OpenWeather API

Technologies

  • HTML
  • CSS
  • JavaScript
  • OpenWeather API
